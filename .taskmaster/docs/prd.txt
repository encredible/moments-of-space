# Moments of Space 웹사이트 PRD (제품 요구사항 문서)

## 프로젝트 개요

**제품명**: Moments of Space (공간의 순간들)
**프로젝트 유형**: 인테리어 디자인 서비스 웹사이트
**대상 사용자**: 인테리어 디자인 서비스를 찾는, 자신의 공간을 개선하고자 하는 개인 및 기업 고객
**개발 플랫폼**: Next.js (React 기반 프레임워크)

## 제품 설명

Moments of Space는 전문적인 인테리어 디자인 서비스를 제공하는 웹사이트입니다. 사용자에게 다양한 인테리어 디자인 서비스를 소개하고, 전후 비교 갤러리를 통해 서비스의 품질을 시각적으로 보여주며, 서비스 문의 및 연락을 위한 통로를 제공합니다.

## 핵심 기능 요구사항

1. **홈페이지**
   - 히어로 섹션: 브랜드와 서비스 소개
   - 서비스 리스트 섹션: 주요 서비스 소개 및 이미지, 자세히 보기 버튼 클릭 시 오버레이 형태로 상세 설명 표시
   - 서비스 하이라이트: 주요 서비스의 강점 강조
   - 인테리어 아이콘 섹션: 다양한 인테리어 스타일 및 요소 소개
   - 전후 비교 슬라이더: 프로젝트 갤러리 형태로 서비스 결과물 시각화
   - 콜투액션(CTA) 섹션: 사용자의 서비스 문의 유도

2. **서비스 페이지**
   - 제공하는 서비스 상세 설명
   - 서비스별 요금 정보
   - 서비스 과정 및 방법론 설명
   - 맞춤형 서비스 제안 가능성 표시

3. **프로젝트 갤러리**
   - 완료된 프로젝트 포트폴리오
   - 전후 비교 기능
   - 프로젝트별 상세 설명
   - 카테고리별 필터링 기능

4. **문의/연락 페이지**
   - 연락처 정보
   - 문의 양식 (이메일 보내기 기능 포함)
   - 카카오톡 상담하기 버튼 및 연동
   - 위치 정보 및 지도
   - 소셜 미디어 링크

## 기술적 요구사항

1. **프론트엔드**
   - Next.js 15.3.3 이상의 최신 버전 사용
   - React 19.0.0 사용
   - 반응형 웹 디자인 (모바일, 태블릿, 데스크톱 대응)
   - 모던하고 미니멀한 UI/UX 디자인
   - Tailwind CSS를 활용한 스타일링
   - 이미지 최적화 및 지연 로딩 구현

2. **성능 최적화**
   - 빠른 페이지 로드 시간 (2초 이내)
   - 코드 분할 및 지연 로딩 구현
   - SEO 최적화
   - 웹 접근성 표준 준수 (WCAG 2.1)

3. **배포 환경**
   - Vercel 플랫폼 활용
   - 지속적 통합 및 배포 파이프라인 구축
   - 환경별 설정 관리 (.env 파일 활용)

## 사용자 경험 목표

1. **직관적인 탐색**
   - 명확한 네비게이션 구조
   - 간결하고 효과적인 정보 제공
   - 일관된 디자인 언어

2. **시각적 매력**
   - 고품질 이미지와 비디오
   - 세련된 애니메이션 및 전환 효과
   - 브랜드 아이덴티티 일관성

3. **신뢰감 구축**
   - 완료된 프로젝트 포트폴리오 강조
   - 고객 후기 및 추천사
   - 전문성 입증 요소

4. **전환율 최적화**
   - 전략적 CTA 배치
   - 간소화된 문의 프로세스
   - 명확한 서비스 가치 제안
